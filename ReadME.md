# Лабораторная работа №1: Конструктивные числа и глобальная оптимизация

## Описание проекта
Реализация типа данных **«конструктивное вещественное число»** (`CReal`) и применение алгоритмов стохастической оптимизации для поиска минимума функции Розенброка. Проект демонстрирует преимущество адаптивной точности при работе со сложными вычислимыми функциями.

## Основные компоненты

### 1. Конструктивные числа (`CReal`)
В отличие от стандартных `f64`, число в данной работе представлено как алгоритм (функция):
* **Принцип**: По запросу точности $n$ число возвращает интервал $[low, high]$, внутри которого оно гарантированно находится.
* **Адаптивное сравнение**: Реализован метод `compare_adaptive`. Если интервалы двух чисел пересекаются, алгоритм автоматически повышает точность $n$ до тех пор, пока не сможет однозначно определить, какое число меньше.
* **Операции**: Реализованы сложение, вычитание, умножение и возведение в квадрат.

### 2. Алгоритмы оптимизации
Реализовано два метода для поиска глобального минимума:
* **Имитация отжига (Simulated Annealing)**:
    * Использует "температуру" для контроля вероятности перехода в худшие состояния.
    * Позволяет выходить из локальных минимумов.
* **Генетический алгоритм (Genetic Algorithm)**:
    * **Арифметическое скрещивание**: создание потомков путем смешивания координат родителей.
    * **Аддитивная мутация**: небольшие случайные отклонения координат для точного поиска на "дне" оврагов.
    * Гарантированное сохранение лучших особей популяции.
